# Carpeta de Imágenes

Coloca aquí tus imágenes para el sitio web.

## Imágenes Recomendadas

### Hero Background (Principal)
- **Archivo**: `hero-background.jpg`
- **Tamaño recomendado**: 1920x1080px o mayor
- **Formato**: JPG o WebP
- **Descripción**: Imagen de camiones, carretera, o logística
- **Ubicación en código**: `src/components/Hero.astro` línea 52

### Galería (6 imágenes)
- **Archivos**: `gallery-1.jpg` a `gallery-6.jpg`
- **Tamaño recomendado**: 800x600px
- **Formato**: JPG o WebP
- **Ubicación en código**: `src/components/Gallery.astro`

### About Section
- **Archivo**: `about-image.jpg`
- **Tamaño recomendado**: 600x700px
- **Formato**: JPG o WebP
- **Ubicación en código**: `src/components/About.astro`

### Features Section
- **Archivo**: `features-image.jpg`
- **Tamaño recomendado**: 600x700px
- **Formato**: JPG o WebP
- **Ubicación en código**: `src/components/Features.astro`

## Cómo Usar las Imágenes

### Opción 1: Background Image (CSS)
```css
background-image: url('/images/tu-imagen.jpg');
```

### Opción 2: Etiqueta IMG (HTML)
```html
<img src="/images/tu-imagen.jpg" alt="Descripción" />
```

## Optimización

Para mejor rendimiento, optimiza tus imágenes:
- Usa formatos modernos (WebP)
- Comprime las imágenes sin perder calidad
- Usa tamaños apropiados (no más grandes de lo necesario)

## Herramientas Recomendadas

- TinyPNG: https://tinypng.com/
- Squoosh: https://squoosh.app/
- ImageOptim: https://imageoptim.com/
